# DX Consulting CRM - 案件管理システム

DXコンサルタント向けの実務的な案件管理システムです。顧客管理、案件進捗管理、エンジニアへの引継ぎ管理を一元化できます。

## 🎯 主な機能

### 1. ダッシュボード
- 今月の新規案件数
- 進行中の案件数
- 見積総額の表示
- エンジニア引継ぎ待ちのアラート
- 最近の活動記録

### 2. 案件管理
- **ご相談内容の記録**: 顧客からの初回相談内容を詳細に記録
- **提案内容・見積金額**: 提案書の内容と見積金額の管理
- **ステータス管理**: 案件の進捗を可視化
  - 初回相談 → ヒアリング中 → 提案作成中 → 見積提示 → 商談中 → エンジニア引継ぎ → 実施中 → 完了
- **案件ごとの進捗履歴**: 活動記録をタイムラインで管理

### 3. 引継ぎ管理
- **@Staff_Notsuさんへの引継ぎ**: 技術的な相談事項の記録
- **浦上泰弘さんへの引継ぎ**: アーキテクチャ設計等の相談記録
- **その他エンジニアへの引継ぎ**: プロジェクトメンバーへの情報共有

### 4. エンジニアバトンタッチ
- 技術的な対応範囲の明確化
- 現在の状況と顧客要件の整理
- スケジュールと予算の管理
- エンジニアの承認状況トラッキング

### 5. 顧客管理
- 顧客企業情報の一元管理
- 業種・企業規模による分類
- 各顧客の案件数表示

### 6. 進捗記録
- 活動種別（打ち合わせ、電話、メール、提案作成など）
- 活動内容と次回アクションの記録
- 記録者の追跡

## 🚀 セットアップ方法

### 1. 必要な環境
- Python 3.11.4以上
- pip
- VisualStudioCodeなど

### 2. インストール

```bash
# 仮想環境の作成
python3 -m venv venv

# 仮想環境の有効化（Linux/Mac）
source venv/bin/activate

# 仮想環境の有効化（Windows）
venv\Scripts\activate

# Djangoのインストール
pip install django

# データベースのマイグレーション
python manage.py migrate

# 管理者ユーザーの作成
python manage.py createsuperuser

# サンプルデータの投入（オプション）
python manage.py load_sample_data
```

### 3. サーバーの起動

```bash
python manage.py runserver
```

ブラウザで以下にアクセス：
- メイン画面: http://127.0.0.1:8000/
- 管理画面: http://127.0.0.1:8000/admin/

## 💡 使い方

### 案件の登録
1. 管理画面 (http://127.0.0.1:8000/admin/) にログイン
2. 「顧客」から新規顧客を登録
3. 「案件」から新規案件を登録
   - 顧客を選択
   - 案件名を入力
   - ご相談内容を記録

### 案件の進捗管理
1. 案件一覧から該当案件を選択
2. 「進捗記録追加」ボタンをクリック
3. 活動種別、内容、次回アクションを記録

### エンジニアへの引継ぎ
1. 案件詳細画面から「エンジニアバトンタッチ記録」を追加
2. エンジニア名、技術対応範囲、予算などを入力
3. エンジニアが承認したら「エンジニア承認済み」にチェック

### 提案書・見積の管理
1. 案件の編集画面を開く
2. 「提案内容」フィールドに提案書の内容を記入
3. 「見積金額」に金額を入力
4. ステータスを「見積提示」または「商談中」に変更

## 🎨 カスタマイズ

### ステータスの追加
`projects/models.py`の`Project.STATUS_CHOICES`を編集

### 引継ぎ先の追加
`projects/models.py`の`Handover.HANDOVER_TYPE_CHOICES`を編集

### 活動種別の追加
`projects/models.py`の`ProgressLog`の`activity_type`のchoicesを編集
